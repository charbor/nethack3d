# nethack3d

A 100% vibecoded 3D dungeon crawler imitating [NetHack](https://en.wikipedia.org/wiki/NetHack), built with Three.js.

<img width="1091" height="559" alt="image" src="https://github.com/user-attachments/assets/0c70b1a7-c4bb-43ef-a738-0218acb48f73" />



## Setup

Run `./serve.sh` and open `localhost:8000` in your browser.
